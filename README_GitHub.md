# المحاسب الأعظم - نظام المحاسبة التجارية
## Al-Muhaseb Al-Azeem - Commercial Accounting System

نظام محاسبة تجارية شامل باللغة العربية مصمم لإدارة الأعمال التجارية بالكامل.

## المميزات

- إدارة المبيعات والفواتير
- إدارة المشتريات والموردين
- إدارة المخزون والمنتجات
- إدارة العملاء والحسابات
- إدارة الموظفين والرواتب
- نظام متعدد الفروع
- تقارير مالية شاملة
- دعم 30 مستخدم
- واجهة عربية RTL

## التثبيت

```bash
# تثبيت المكتبات
npm install

# تشغيل النظام
npm run dev
```

## التقنيات المستخدمة

- Frontend: React + TypeScript + Tailwind CSS
- Backend: Node.js + Express
- Database: PostgreSQL (Neon)
- ORM: Drizzle

## المتطلبات

- Node.js 18+
- PostgreSQL Database
